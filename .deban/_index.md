# BJJ_StagesGoalsTools — Index

## Brief
Unpack key BJJ positions away from linear progressions. Map them as constantly moving checklists of intermediary goals that must be cleared at various stages — not always in order — where the practitioner has access to a variety of tools at each stage. Two existing static diagrams (Back Attacks from Danaher, Butterfly Attacks from Marcelo Garcia) serve as the reference examples to reconstruct as interactive diagrams.
